## Введение

Цель работы - проверить статистические гипотезы по данным варианта C-6. В работе используются параметрический критерий Стьюдента для двух независимых выборок, критерий χ² для проверки гипотезы о дисперсии нормального распределения, непараметрический критерий Манна-Уитни и критерий согласия Пирсона χ².

Практический контекст группы связан с играми и дизайном. Столбцы X1 и X2 интерпретируются как время обработки кадра в двух режимах работы системы, X3 - как результаты измерения одного параметра производительности, X4 - как интервалы времени между событиями в системе. Все величины измеряются в миллисекундах.

Отчёт построен так, чтобы им можно было пользоваться при защите: в каждом разделе указаны формулировки гипотез, основания выбора критерия, используемые формулы, происхождение всех чисел, вычисление статистики, правило принятия решения и содержательная интерпретация результата.

## 1. Исходные данные и условия варианта

Вариант: C-6.

Объём выборки: n = 105.

Уровень значимости: α = 0,05.

По карточке варианта необходимо выполнить четыре проверки:

- по столбцам X1 и X2 проверить гипотезу о равенстве математических ожиданий двух независимых нормальных совокупностей;
- по столбцу X3 проверить гипотезу H0: σ² = 3,54;
- по тем же столбцам X1 и X2 применить непараметрический критерий для двух независимых выборок;
- по столбцу X4 проверить гипотезу H0 о том, что X4 имеет показательное распределение с параметром λ = 0,117, используя критерий согласия Пирсона χ².

Если в карточке варианта не указано иное, используется двусторонняя альтернатива. Это соответствует требованиям методических материалов по РГР №2.

## 2. Краткая теоретическая основа работы

Проверка статистической гипотезы всегда начинается с формулировки нулевой гипотезы H0 и альтернативной гипотезы H1. Далее выбирается статистика критерия h, вычисляется её наблюдаемое значение hнабл и по нему принимается решение: отвергать H0 или нет.

Уровень значимости α - это вероятность ошибки первого рода. Ошибка первого рода означает, что нулевая гипотеза на самом деле верна, но по результатам проверки мы её отвергаем.

Если используется p-value, правило решения имеет вид: если p-value < α, то нулевая гипотеза отвергается; если p-value ≥ α, то оснований отвергнуть H0 нет.

Для двух независимых нормальных выборок X1 и X2 с неизвестными, но предполагаемо равными дисперсиями применяется двухвыборочный критерий Стьюдента. Его статистика имеет вид:

$$
t = \frac{\overline{X}_1 - \overline{X}_2}{S_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}
$$

где:

$$
S_p^2 = \frac{(n_1 - 1)S_1^2 + (n_2 - 1)S_2^2}{n_1 + n_2 - 2}
$$

Для проверки гипотезы о дисперсии нормального распределения используется статистика:

$$
\chi^2 = \frac{(n - 1)S^2}{\sigma_0^2}
$$

которая при справедливости H0 имеет распределение χ² с n - 1 степенями свободы.

Для сравнения двух независимых выборок без предположения о нормальности применяется критерий Манна-Уитни. Он основан не на самих значениях, а на их рангах в объединённой выборке.

Для проверки согласия распределения X4 с заданным законом используется критерий Пирсона χ². Выборка разбивается на интервалы, для каждого интервала считаются наблюдаемая частота nk и ожидаемая частота npk, после чего вычисляется сумма:

$$
\chi^2 = \sum \frac{(n_k - np_k)^2}{np_k}
$$

## 3. Проверка гипотезы о равенстве математических ожиданий по X1 и X2

В содержательном смысле здесь проверяется, одинаково ли в среднем время обработки кадра в двух режимах работы системы.

Нулевая гипотеза: H0: M(X1) = M(X2).

Альтернативная гипотеза: H1: M(X1) ≠ M(X2).

Выбор критерия. По условию задачи речь идёт о двух независимых нормальных совокупностях. В методических материалах для такого случая приведён двухвыборочный критерий Стьюдента при неизвестных, но предполагаемо равных дисперсиях. Поэтому именно этот критерий используется в решении.

Ошибка первого рода в данной задаче состоит в том, что мы сделаем вывод о различии средних времён обработки кадра в двух режимах, хотя в действительности средние равны.

| Показатель | Для X1 | Для X2 |
| --- | ---: | ---: |
| Объём выборки | 105 | 105 |
| Выборочное среднее X̄ | 10.9876 | 13.6128 |
| Выборочная дисперсия S² | 7.4642 | 8.3567 |
| Выборочное стандартное отклонение S | 2.7321 | 2.8908 |
| Минимум / максимум | 4.85 / 20.17 | 5.11 / 21.87 |

Откуда берутся эти числа. Выборочное среднее вычисляется как сумма всех наблюдений, делённая на их количество:

$$
\overline{X}_1 = \frac{x_{11} + x_{12} + \ldots + x_{1,105}}{105}
$$

Выборочная дисперсия S² считается по формуле:

$$
S^2 = \frac{\sum (x_i - \overline{X})^2}{n - 1}
$$

Деление на n - 1 используется для несмещённой оценки дисперсии.

После подстановки чисел получаем объединённую оценку дисперсии:

$$
S_p^2 = \frac{(105 - 1)\cdot 7.4642 + (105 - 1)\cdot 8.3567}{105 + 105 - 2} = 7.9104
$$

Тогда наблюдаемое значение статистики равно:

$$
t_{\text{набл}} = \frac{10.9876 - 13.6128}{\sqrt{7.9104\left(\frac{1}{105} + \frac{1}{105}\right)}} = -6.7629
$$

Число степеней свободы:

$$
k = 105 + 105 - 2 = 208
$$

Критическое значение для двусторонней альтернативы при α = 0,05:

$$
t_{\text{кр}} = t(0{,}975; 208) = 1.9714
$$

Так как |tнабл| = 6.7629 > 1.9714, нулевая гипотеза отвергается. То же решение даёт и p-value: p-value = 1.34339e-10 < 0,05.

Итоговый вывод по пункту 1: на уровне значимости 0,05 имеются статистически значимые основания считать, что среднее время обработки кадра в двух режимах различается. По выборочным средним видно, что режим X2 в среднем даёт большее время обработки кадра, чем режим X1.

## 4. Проверка гипотезы о дисперсии по X3

В этом пункте по карточке варианта проверяется гипотеза о дисперсии нормального распределения.

Нулевая гипотеза: H0: σ² = 3,54.

Альтернативная гипотеза: H1: σ² ≠ 3,54.

Выбор критерия. Для нормального распределения при неизвестном математическом ожидании и проверке гипотезы о дисперсии используется критерий χ².

Ошибка первого рода состоит в том, что будет сделан вывод о том, что дисперсия не равна 3,54, хотя в действительности она равна 3,54.

По выборке X3 получаем: X̄3 = 16.0608, S² = 2.9185, S = 1.7084.

Статистика критерия:

$$
\chi^2_{\text{набл}} = \frac{(n - 1)S^2}{\sigma_0^2} = \frac{(105 - 1)\cdot 2.9185}{3{,}54} = 85.7412
$$

Число степеней свободы:

$$
k = n - 1 = 104
$$

Для двусторонней альтернативы критическая область имеет вид χ² < χ²(0,025; 104) или χ² > χ²(0,975; 104).

Численно:

$$
\chi^2(0{,}025; 104) = 77.6722,\quad \chi^2(0{,}975; 104) = 134.1112
$$

Так как 77.6722 < 85.7412 < 134.1112, наблюдаемое значение статистики не попадает в критическую область. Следовательно, оснований отвергать H0 нет.

То же решение видно по p-value: p-value = 0.1930 > 0,05.

Итоговый вывод по пункту 2: на уровне значимости 0,05 данные не противоречат гипотезе о том, что дисперсия показателя производительности X3 равна 3,54.

## 5. Непараметрический критерий для X1 и X2

В этом пункте используется критерий Манна-Уитни. Он нужен как непараметрическая альтернатива сравнению двух независимых выборок. Его преимущество в том, что он не требует строгого предположения о нормальности и работает через ранги.

Нулевая гипотеза: H0: распределения X1 и X2 совпадают.

Альтернативная гипотеза: H1: распределения X1 и X2 различаются.

Ошибка первого рода состоит в том, что мы заключим, что распределения X1 и X2 различаются, хотя в действительности различий нет.

Откуда берётся статистика U. Сначала обе выборки объединяются в одну общую выборку объёма 210. Затем все значения ранжируются по возрастанию. После этого суммируются ранги элементов одной из выборок, и из этой суммы строится статистика U.

По данным варианта получено:

```text
Uнабл = 2732.5,
p-value = 2.73837e-10.
```

Так как p-value < 0,05, нулевая гипотеза отвергается.

Итоговый вывод по пункту 3: непараметрический критерий также показывает статистически значимые различия между выборками X1 и X2.

Сравнение с пунктом 1. Выводы параметрического и непараметрического критериев совпали. Это хороший признак устойчивости результата: различие между режимами проявляется и при сравнении средних, и при ранговом сравнении выборок.

## 6. Проверка согласия распределения X4 с показательным законом

В этом пункте проверяется, можно ли считать интервалы времени между событиями в системе показательными с заданным параметром λ = 0,117.

Нулевая гипотеза: H0: X4 имеет показательное распределение с параметром λ = 0,117.

Альтернативная гипотеза: H1: X4 не имеет такого распределения.

Выбор критерия. По условию требуется использовать критерий согласия Пирсона χ².

Ошибка первого рода состоит в том, что мы отвергнем гипотезу о показательном распределении с параметром 0,117, хотя на самом деле это распределение подходит.

Чтобы критерий Пирсона применялся корректно, ожидаемые частоты в интервалах не должны быть слишком малы. Удобно разбить область значений не на равные по длине, а на интервалы с одинаковой теоретической вероятностью попадания. В отчёте использованы 10 интервалов, каждый из которых при H0 имеет вероятность 0,1. Тогда ожидаемая частота в каждом интервале равна n * 0,1 = 10,5.

Откуда берутся границы интервалов. Для показательного распределения функция распределения имеет вид F(x) = 1 - e^(-λx). Если положить F(xk) = pk, то:

$$
x_k = \frac{-\ln(1 - p_k)}{\lambda}
$$

При pk = 0,1; 0,2; ...; 0,9 получаем границы интервалов.

| № | Интервал | nk | npk | Вклад в χ² |
| ---: | --- | ---: | ---: | ---: |
| 1 | [0.000; 0.901) | 8 | 10.5 | 0.5952 |
| 2 | [0.901; 1.907) | 8 | 10.5 | 0.5952 |
| 3 | [1.907; 3.049) | 4 | 10.5 | 4.0238 |
| 4 | [3.049; 4.366) | 8 | 10.5 | 0.5952 |
| 5 | [4.366; 5.924) | 10 | 10.5 | 0.0238 |
| 6 | [5.924; 7.832) | 9 | 10.5 | 0.2143 |
| 7 | [7.832; 10.290) | 14 | 10.5 | 1.1667 |
| 8 | [10.290; 13.756) | 13 | 10.5 | 0.5952 |
| 9 | [13.756; 19.680) | 7 | 10.5 | 1.1667 |
| 10 | [19.680; +∞) | 24 | 10.5 | 17.3571 |

Сумма вкладов по всем интервалам даёт наблюдаемое значение статистики:

$$
\chi^2_{\text{набл}} = 26.3333
$$

Так как параметр λ задан в условии и не оценивается по выборке, число степеней свободы равно m - 1, где m - число интервалов. Здесь m = 10, поэтому k = 9.

Критическое значение:

$$
\chi^2_{\text{кр}} = \chi^2(0{,}95; 9) = 16.9190
$$

Так как χ²набл = 26.3333 > 16.9190, нулевая гипотеза отвергается.

То же решение даёт p-value = 0.0018 < 0,05.

Итоговый вывод по пункту 4: на уровне значимости 0,05 данные по X4 не согласуются с показательным распределением с параметром λ = 0,117.

## 7. Итоговые выводы

По столбцам X1 и X2 была проверена гипотеза о равенстве математических ожиданий двух независимых нормальных совокупностей с помощью двухвыборочного критерия Стьюдента. Нулевая гипотеза отвергнута: средние значения статистически различаются.

По столбцу X3 была проверена гипотеза о дисперсии нормального распределения H0: σ² = 3,54 с помощью критерия χ². Нулевая гипотеза не отвергнута: данные согласуются с указанным значением дисперсии.

Для X1 и X2 дополнительно применён непараметрический критерий Манна-Уитни. Он также отвергнул нулевую гипотезу, поэтому вывод о различии двух режимов является устойчивым.

По столбцу X4 с использованием критерия согласия Пирсона проверялась гипотеза о показательном распределении с параметром λ = 0,117. Нулевая гипотеза отвергнута: данная модель не описывает выборку удовлетворительно.

Содержательно это означает, что два режима обработки кадра статистически различаются по времени работы, параметр производительности X3 имеет разброс, совместимый с заданной дисперсией, а интервалы между событиями X4 не подчиняются заявленному показательному закону.

## Приложение Б. Ответы на типовые вопросы на защите

**Что такое уровень значимости α?**

Уровень значимости - это заранее выбранная вероятность ошибки первого рода. В данной работе α = 0,05.

**Что такое ошибка первого рода?**

Ошибка первого рода возникает, если нулевая гипотеза верна, но по результатам критерия она отвергается.

**Почему для X1 и X2 применялся критерий Стьюдента?**

По условию проверяются математические ожидания двух независимых нормальных совокупностей. При неизвестных, но предполагаемо равных дисперсиях применяется двухвыборочный критерий Стьюдента с объединённой оценкой дисперсии.

**Почему дополнительно использован критерий Манна-Уитни?**

Критерий Манна-Уитни является непараметрическим и не требует строгого предположения о нормальности. Он позволяет проверить устойчивость вывода о различии двух выборок.

**Почему в критерии согласия для X4 степеней свободы 9?**

Использовано 10 интервалов, а параметр λ задан в условии и не оценивается по выборке. Поэтому число степеней свободы равно m - 1 = 10 - 1 = 9.

**Какой общий результат работы?**

Средние значения X1 и X2 статистически различаются, дисперсия X3 не противоречит значению 3,54, а распределение X4 не согласуется с показательным законом с параметром λ = 0,117.
