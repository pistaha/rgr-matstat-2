# rgr-matstat

Полноценный Python-проект для РГР по математической статистике. Репозиторий автоматизирует загрузку данных, описательную статистику, оценивание параметров распределений, доверительные интервалы и построение графиков.

## Возможности
- анализ нескольких столбцов данных;
- оценки параметров для равномерного, нормального и сдвинутого экспоненциального распределений;
- сравнение метода моментов и ММП;
- доверительные интервалы;
- сохранение гистограмм и ECDF в `output/`.

## Стек
- Python
- pandas / numpy / scipy / matplotlib

## Запуск локально
```bash
python3 -m pip install -r requirements.txt
python3 main.py
```
